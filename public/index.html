✨ Features
🎫
Token Tracker
Add, manage, and track tokens with charges. Filter by date, agent or keyword. Active and Completed tabs with live counts.
💰
Financial Dashboard
Real-time Total Due calculation on active tokens. Export filtered data to CSV for accounting and record-keeping.
📄
KYC Document Upload
Admin can upload KYC images to Cloudinary after token completion. Agent receives instant notification to view their KYC document.
💬
Discussion Board
Real-time team chat replacing WhatsApp. Multiple channels, token alert notifications, unread badges, and message history.
✉️
Direct Messages
Private messaging between Admin and Agents. Agents can only message Admin. Admin can message each agent individually.
🔐
Role-Based Access
Admin and Agent roles with granular permissions. Agents manage their own tokens; Admin oversees all. Secure session-based login.
⚡
Real-Time Updates
Powered by Socket.IO for instant message delivery, KYC notifications, and live token alerts across all connected users.
📦
Backup & Restore
Full database backup to JSON with one click. Selectively restore tokens, messages, or user data from any previous backup file.
🔧
Admin Controls
Add/remove agents, reset passwords, view system statistics, manage channels, and maintain complete oversight of all operations.
☁️
Cloud Ready
Deployed on Railway with PostgreSQL. Cloudinary for KYC image storage. Accessible from anywhere on any device.
